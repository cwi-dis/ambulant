Ambulant Design Documentation

Currently the design documentation consists of the following bits and pieces:

  • Overall has a description of the design documentation, the source code organisation, source code conventions and some high-level design decisions.
  • Objects has descriptions of the various classes used in Ambulant. Accompanying UML graphics can be found in the models subdirectory.

Two older documents that have not been integrated yet, and that may not be up-to-date are remarks.txt and sdd-schema.txt.
