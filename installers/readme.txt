Here are some scripts to help create various distributions.
Some need manual attendance, some don't.
Some have some variables that need to be changed, so don't try
to run these without inspecting them.

mkdebiandist.sh    Create a .deb package, tested on Ubuntu 10.10
nsis-win32/        Create a NullSoft Installer on Windows
sh-macos/          Create a MacOSX DMG installer
