This directory contains the Ambulant WebKit plugin for Firefox
on Linux distr. Fedora Core 8/ 64bit

This plugin allows Firefox and other Mozilla-based browsers to play back
SMIL content in a web page.


********** linux: KB TMP. instructions how to build and run (local on bahco.sen.cwi.nl):

- build:
PKG_CONFIG_PATH=:/usr/lib64/pkgconfig/ ./autogen.sh --prefix=$HOME \
--without-xerces --without-xerces-plugin --without-live \
--with-firefox-plugin CFlAGS="-g -O0 -fPIC" CXXFLAGS="-g -O0 -fPIC";make

- install:
cd src/firefox_plugin_gtk;cp libAmbulantPlugin.xpt libAmbulantPlugin.so ~/.mozilla/plugins/

- Restart Firefox:
export
LD_LIBRARY_PATH=/export/scratch1/Ambulant/ambulant/third_party_packages/mozilla/ff-opt-static/dist/bin/
MOZ_PLUGIN_PATH=~/.mozilla/plugins /usr/lib64/firefox-2.0.0.12/firefox-bin

************ windows: KB TMP. 201207

Sometimes it helps by setting
export MOZ_PLUGIN_PATH=<path-to-your-pugin-path>,
e.g. ~/.mozilla/plugins before starting firefox

  
There is one known issue with the plugin: relative pathnames in the
<embed src="..."> tag do not work, please use absolute pathnames only.

More information on the Ambulant Player and plugin can be found via
<http://www.ambulantPlayer.org>
