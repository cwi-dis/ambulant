Installation instructions for installing the AmbulantPlayer Firefox plugin
version 2.0.2 (xpcom version, works only in Firefox) on Windows XP:
Move or copy the following files from this .zip file to:
C:\Program Files\Mozilla Firefox\plugins

avcodec-51.dll
avdevice-52.dll
avformat-52.dll
avutil-49.dll
npambulant.dll
npambulant.xpt
libambulant-shwin32.dll
libamplugin_plugin.dll
libamplugin_state_xpath.dll
libamplugin_ffmpeg.dll
libamplugin_plugin.dll
xerces-c_2_8.dll
SDL.dll

For uninstall, remove above files.

WARNING: this version of the AmbulantPlayer Firefox plugin may be incompatible with
some other plugins, e.g. Greasemonkey.
