Ambulant Plugin Example for Mozilla Test Case

This test for the AmbulantPlayer Mozilla plugin shows the basic seamless playback test. It implicitly also tests Ambulant plugin loaded from a Firefox plugin, smilState and smilText. You should see the AmbulantPlayer plugin window playing the sptest-00 document. All features in SMIL 3.0 are supported.

 

The purpose of the sptest-00 demo is to test seamless playback operation. This version here uses SMIL Text for timed text animation, and smilState to get the video files.
